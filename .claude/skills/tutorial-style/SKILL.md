# Tutorial Style

Use this skill when writing beginner-facing guides, walkthroughs, or setup docs.

## Workflow

1. Define unfamiliar terms before using them.
2. Prefer short steps with exact commands.
3. Say what the user should expect to see after each step.
4. Avoid assuming prior CLI or editor knowledge.
